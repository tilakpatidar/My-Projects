Introduction:

Shows images in the gallery when a image is updated in DropBox folder.


**********************************************************************
Instructions:

Copy imgResult.py in your cgi-bin from cgi-bin folder present within project.
Path given in html file is /cgi-bin/imgResult.py.
Change it according to your need.
Also change the DropBox folder location in imgResult.py to your DropBox folder location.
